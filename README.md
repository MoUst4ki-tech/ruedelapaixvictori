## Project Structure

- `jam-game/`: The Next.js + Phaser game source code.
- `setup-dev.sh`: Script to set up the development environment on Linux.
- `setup-dev-mac.sh`: Script to set up the development environment on macOS.

## Getting Started

1. **Setup Environment** (if needed):
   
   **Linux:**
   ```bash
   ./setup-dev.sh
   ```

   **macOS:**
   ```bash
   ./setup-dev-mac.sh
   ```

2. **Run the Game**:
   ```bash
   cd jam-game
   npm install
   npm run dev
   ```
